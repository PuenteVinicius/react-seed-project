# React Seed Project #
  
  This a seed for react project. This contains:

    - redux
    - redux thunk
    - redux form
    - redux multi
    - redux router
    - redux promise 
    - Jest with enzyme
    - material-ui

# Running the project #

  In this project we are using yarn so:

    $ yarn install

    $ yarn start

# Building the project #

    $ yarn build

# Running testes #

  running all tests:

      $ yarn test

  See test coverage:      

      $ yarn test --coverage
